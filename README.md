Repositório criado para estudos e prática da linguagem Java.  
Contém exemplos básicos de sintaxe, orientação a objetos e conceitos fundamentais.  
Os códigos são focados em aprendizado progressivo e entendimento da linguagem.  
Inclui classes, métodos, construtores e uso de objetos.  
Objetivo principal é fortalecer a base em Java para aplicações back-end.  
Os exemplos priorizam clareza e simplicidade.  
Sem frameworks, apenas Java puro.  
Pode conter testes e pequenos experimentos.  
Evolução contínua conforme avanço nos estudos.  
Uso livre para consulta e aprendizado.
